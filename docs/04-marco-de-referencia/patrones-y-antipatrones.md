## Patrones

- Gestión centralizada de secretos
- Uso de identidades por workload
- Infraestructura como código
- Automatización de despliegues
- Separación de responsabilidades

## Antipatrones

- Uso de secretos embebidos en código
- Uso de tokens estáticos compartidos
- Permisos excesivos
- Configuración manual fuera de Git
- Confianza implícita basada en red
